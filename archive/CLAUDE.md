# Project Goal

Building a general-purpose agent able to local files, create docs, PPT creation, Excel analysis, and perfroming command execution and moving and managing the device files. All using local llms only. The idea is to push the agents to be able to work within files and folders locally while only using simple tools like:

- FileToolkit : for reading, writing, editing, searching
  - Read files
  - Write files
  - Edit files 
  - Glob
  - Grep 
  - & bash commands
  - Online search (duckduckgo future integration)

# Code and Doc details
I am using agno to create an agent that is able to access a terminal and write code using only local llm models from ollama and in the future vllm and sglang as llm source.
