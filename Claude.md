# Project Goal

Building a general-purpose agent able to access emails, Notion, local files, doc creation, PPT creation, Excel analysis, and perfroming command execution and moving and managing the device files.

# Project Guide

Use AGNO Python Agentic AI library with a local SQLite database in the tmp folder. All openrouter api keys and future api keys will be in .env( you are never allowed to access it).
Use the https://docs.agno.com/llms.txt for any details about the docs in AGNO. Check for syntax issues before executing any code. Use the test folder as the main folder for the code-assistant agent to use for all of its work.
